ESP32와 LLM을 활용한 음성 대화형 스마트 미러 설계 및 구현

Design and Implementation of a Voice-Interactive Smart Mirror Using ESP32 and LLM
정재석 · 지민우 · 윤여명 · 장준혁
한남대학교 컴퓨터공학과  |  교신저자: 장준혁 (jhjang@hnu.kr)
요약 (Abstract)
본 논문은 ESP32-S3와 Arduino UNO의 이중 마이크로컨트롤러 구조를 기반으로, 음성 인식·LLM 처리·TTS 출력이 결합된 스마트 미러 시스템을 설계 및 구현하였다. PIR 센서로 사용자 접근 시에만 시스템이 활성화되며, 호출어 포함 시에만 Gemini 2.5 Flash를 통해 실시간 답변을 생성한다. LED 및 LCD 기반 멀티모달 피드백으로 체감 대기시간을 단축하고, 스마트 홈 허브로서의 활용 가능성을 검증하였다.
스마트 미러 ESP32 LLM 음성 인터페이스 IoT Gemini 2.5 STT / TTS
배경 및 목표
문제 정의
  • 고가·폐쇄 플랫폼 문제로 기존 스마트 미러의 일반 가정 접근성이 낮음
  • 정전식 터치스크린은 욕실 습기·지문 오염 등 물리적 단점 존재
  • 기존 제품은 단방향 정보 표시에 한정 — 양방향 음성 상호작용 부재
제안 솔루션
  • 오픈소스 MCU(ESP32 + Arduino UNO) 기반 저비용 구축
  • STT → LLM → TTS 파이프라인으로 양방향 음성 대화 구현
  • PIR 센서 기반 비접촉 인터페이스로 물리적 한계 극복

주요 최적화
  • 호출어 "미러" 포함 시에만 LLM 전송 → API 비용 절감
  • 답변 80자 이내 제한 + 날씨·시간 컨텍스트 주입으로 응답 최적화
  • WDT 피딩으로 블로킹 구간 강제 재부팅 방지
  • TCP/SSL 정리 지연 + 수신 버퍼 확장 → 오디오 끊김 없는 TTS
상태 동기화 & 시각적 피드백
음성 인터페이스의 상태 불확실성을 해소하기 위해,
ESP32가 처리 상태를 UART로 전송하면 Arduino UNO가
LED 색상과 LCD 텍스트를 실시간으로 변경한다.

소프트웨어 안정성 최적화
임베디드 환경에서의 안정적인 AI 서비스 구동을 위해
다음 세 가지 소프트웨어 최적화를 수행하였다.
  • 저전압 감지 비활성화 — 스피커 출력 시 순간적인 전압 강하로
    인한 강제 리셋을 방지하기 위해 LVD 기능을 비활성화하고
    오디오 볼륨을 최적화하여 전력 피크를 제어
  • 워치독 타이머(WDT) 피딩 — 네트워크 통신·오디오 재생 등
    블로킹 구간에 WDT 피딩 로직을 삽입하여
    오동작으로 인한 강제 재부팅 방지
  • TTS 스트리밍 안정화 — TCP/SSL 네트워크 스택 정리를 위한
    지연 부여 및 수신 버퍼 타임아웃 확장으로 네트워크 지연 상황에서도
    오디오 끊김 없는 자연스러운 음성 합성 구현
